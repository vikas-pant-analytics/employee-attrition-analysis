# Employee Attrition Analysis

## Project Overview
This project investigates employee attrition using Excel and SQL. The goal was to identify root causes of employee turnover and design data-driven retention strategies.

## Objectives
- Analyze factors contributing to employee attrition
- Test salary, satisfaction, tenure, and training impact
- Identify high-risk employee segments
- Provide actionable HR recommendations

## Tools Used
- Excel (data cleaning & feature engineering)
- SQL (analysis & risk modeling)
- GitHub (project documentation)

## Key Findings
- Lower salary strongly correlates with attrition
- Early-tenure employees face higher risk
- Satisfaction levels influence retention
- Attrition hotspot identified in R&D department

## Recommendations
- Strengthen early-career support programs
- Expand training and growth opportunities
- Improve satisfaction and workload balance
- Focus targeted intervention in R&D
